# 深度学习与智能驾驶实践工坊

## README 摘要

本仓库围绕“深度学习与智能驾驶实践工坊”教学闭环组织，教程覆盖深度学习基础、CIFAR-10 小数据对比实验、智能驾驶感知/预测/决策实操（行为克隆、BEV 表示、轨迹预测、VLA 体验）等内容，并配套可运行脚本与可视化产出。建议从核心教程按章节顺序阅读与实践。

## 快速开始

- 环境准备：建议使用本地 Python 虚拟环境（可自行创建）
- 进入教程：从 `易精智能驾驶算法与编程实践工坊教程_v2.2.md` 开始
- 关键脚本入口：`tutorial_runs/`

## 目录说明

- CIFAR-10 实验脚本与输出（CNN vs ResNet18）

- 轨迹预测训练与可视化
- VLA 语言引导决策演示

# 易精智能驾驶算法与编程实践工坊课程大纲

### 第一阶段 环境搭建与工具链
快速搭建Python深度学习环境，验证PyTorch可用性，深度学习的基础实战。配置AI辅助编程环境（GitHub Copilot/Cursor等），提升课堂实操效率。

### 第二阶段 智能驾驶知识体系
建立"感知→预测→决策→控制"完整闭环体系，梳理感知、世界模型、BEV/3D、VLM/VLA等前沿技术，理解Tesla FSD、李飞飞世界模型等工业方案演进逻辑。

### 第三阶段 智能驾驶案例实操
基于开源驾驶数据集，从端到端到结构化感知，从单模态到多模态融合：
- **行为克隆**：端到端学习，图像→转向角直接映射
- **世界模型**：车辆轨迹的时序预测，掌握状态转移建模
- **BEV表示**：图像→俯视图，体验度量空间感知优势
- **VLA决策**：视觉-语言-动作融合，探索多模态决策前沿
（案例暂定）

### 第四阶段 趋势前瞻与职业发展
技术演进趋势：端到端回归（Tesla FSD v12）、世界模型与BEV融合、多模态大模型、具身智能。聚焦"技术能力地图"，从算法工程师到智驾全栈专家，构建核心能力，思考终身职业规划。
